NTC\/260: Foundations of Cloud Services View course details 3 Credits This course introduces practices in cloud infrastructure and services. Covered are a cloud computing model, layers of cloud computing, and functions of cloud computing. Included are fundamental layers of cloud infrastructure: physical, virtual, control, orchestration, and service. Also included are cross-layer functions of cloud infrastructure: business continuity, security, and service management. CYB\/225: Linux Fundamentals View course details 3 Credits This is an introductory course on Linux that provides sufficient command of the subject to meet the initial needs of penetration testing. NTC\/300: Cloud Technologies View course details 3 Credits This course will provide the knowledge and skills required to understand foundational and advanced cloud terminologies\/methodologies; to implement, maintain, and deliver cloud technologies and infrastructures (e.g., server, network, storage, and virtualization technologies); and to understand aspects of IT security and use of industry best practices related to cloud implementations. NTC\/302: Network Web Services View course details 3 Credits This course focuses on Network Web Services with a specific focus on Amazon Web services (AWS CCP). The courses will help an IT professional gain knowledge and skills necessary to effectively demonstrate an overall understanding of the AWS Cloud, independent of specific technical roles addressed by other AWS Certifications. NTC\/304: Cloud Developer View course details 3 Credits This course introduces developing on Amazon Web Services (AWS), including understanding AWS services, its uses, and the AWS architecture. Covering foundational knowledge on how to configure, develop, deploy and debug secure cloud-based applications as well as best practices when using AWS. NTC\/306: Cloud Operations View course details 3 Credits This course focuses on deploying, managing and operating systems on Amazon Web Services (AWS).
